# Template de Acompanhamento — DegustIA

> Planilha de horas recuperadas
> Recomenda-se copiar para o Google Sheets ou Notion para preencher semanalmente.

---

## Instruções

Preencha uma linha por ferramenta/prática recomendada. A cada semana, registre quantos minutos por semana aquela tarefa está tomando. O objetivo é ver a **curva de redução** ao longo do tempo.

## O que fazer com esse template

- **Mês 1**: preencher semanalmente (5 min por semana)
- **Mês 2**: preencher só se algo mudou
- **Mês 3 em diante**: usar só pra novas recomendações

O objetivo não é controlar pra sempre — é ter dados pra decidir se a ferramenta realmente liberou tempo. Depois que a economia se estabiliza, o template cumpriu seu papel.
